📝 Instrucciones de prueba:

  1. Abre el DevTools del navegador (F12)
  2. Activa el modo responsive (Ctrl+Shift+M o Cmd+Shift+M)
  3. Prueba estos anchos clave:
    • 320px - Mobile Small (iPhone 5)
    • 360px - Mobile Standard (Galaxy S8) ← MAYORÍA
    • 375px - Mobile Standard (iPhone 7)
    • 414px - Mobile Large (iPhone 8 Plus)
    • 768px - Tablet (iPad)
    • 1024px - Desktop
    • 1920px - Large Desktop
  4. Observa cómo cambian los tamaños de los elementos
  5. El indicador superior derecha muestra el breakpoint activo
  6. Los valores de las variables se actualizan en tiempo real arriba

💡 Nota: Este demo simula el sistema de variables de responsive-variables.scss. Los valores mostrados aquí coinciden con los definidos en el archivo SCSS del proyecto.
