# Documentación 3.5c - Contenedores (Assets)

## Fecha: 2025-12-05

## Descripción General
Implementación completa del sistema de contenedores visuales usando assets PNG para todos los displays de información del juego.

## Cambios Implementados

### 1. Sistema de Contenedores de Imágenes

#### Archivos de Assets Utilizados
- `assets/images/contenedores/saldo.png` - Contenedor para el balance del jugador
- `assets/images/contenedores/apuesta.png` - Contenedor para apuestas individuales
- `assets/images/contenedores/apuesta-total.png` - Contenedor para apuesta total
- `assets/images/contenedores/panel-apuestas.png` - Contenedor de fondo del panel de apuestas
- `assets/images/contenedores/aro-dorado.png` - Marco contenedor para imagen del animal seleccionado
- `assets/images/contenedores/nombre-animal.png` - Contenedor para el nombre del animal seleccionado

#### Ubicaciones Implementadas

**Archivo**: [home.page.html](src/app/home/home.page.html)

1. **Balance del Jugador** (líneas 9-12)
   - Ubicación: Header superior derecho
   - Contenedor: `saldo.png`
   - Display: Balance actual del jugador
   ```html
   <div class="player-balance-display">
     <img src="assets/images/contenedores/saldo.png" alt="Saldo" class="balance-container-image">
     <div class="balance-value">${{ balance$ | async }}</div>
   </div>
   ```

2. **Apuesta Actual** (líneas 65-68)
   - Ubicación: Panel central de apuestas
   - Contenedor: `apuesta.png`
   - Display: Monto apostado al animal seleccionado
   ```html
   <div class="bet-amount-display">
     <img src="assets/images/contenedores/apuesta.png" alt="Tu Apuesta" class="bet-container-image">
     <div class="bet-value">${{ (selectedAnimals | findBet:currentEditingAnimal)?.amount || 0 }}</div>
   </div>
   ```

3. **Apuesta Total** (línea 80)
   - Ubicación: Panel flotante de balance
   - Contenedor: `apuesta-total.png`
   - Display: Suma de todas las apuestas activas

4. **Panel de Apuestas** (línea 52)
   - Ubicación: Contenedor de controles de apuesta (betting-panel)
   - Contenedor: `panel-apuestas.png`
   - Display: Fondo del panel con fichas, animal seleccionado y botones
   ```html
   <div class="betting-panel">
     <img src="assets/images/contenedores/panel-apuestas.png" alt="Panel de Apuestas" class="betting-panel-background">
     <!-- Contenido del panel -->
   </div>
   ```

5. **Aro Dorado - Animal Seleccionado** (línea 47)
   - Ubicación: Contenedor del animal en el panel central
   - Contenedor: `aro-dorado.png`
   - Display: Marco dorado alrededor de la imagen del animal
   ```html
   <div class="animal-image-container" *ngIf="currentEditingAnimal">
     <img src="assets/images/contenedores/aro-dorado.png" alt="Aro Dorado" class="animal-container-frame">
     <img [src]="currentEditingAnimal.image" alt="{{ currentEditingAnimal.name }}" class="selected-animal-image">
   </div>
   ```

6. **Nombre del Animal** (línea 43)
   - Ubicación: Tag sobre el animal en el panel central
   - Contenedor: `nombre-animal.png`
   - Display: Etiqueta con el nombre del animal seleccionado
   ```html
   <div class="animal-name-tag" *ngIf="currentEditingAnimal">
     <img src="assets/images/contenedores/nombre-animal.png" alt="Nombre Animal" class="animal-name-container-image">
     <span class="animal-name-text">{{ currentEditingAnimal.name }}</span>
   </div>
   ```

### 2. Estructura CSS

**Archivo**: [home.page.scss](src/app/home/home.page.scss)

#### Clases de Contenedores
```scss
.balance-container-image,
.bet-container-image,
.pill-container-image {
  position: absolute;
  width: 100%;
  height: 100%;
  object-fit: contain;
  z-index: 1;
}
```

#### Displays de Valores
```scss
.balance-value,
.bet-value,
.pill-value {
  position: relative;
  z-index: 2;
  font-size: [variable según contexto];
  color: var(--gold-primary);
  text-shadow: 0 0 10px rgba(255, 215, 0, 0.5);
}
```

### 3. Sistema de Posicionamiento

Todos los contenedores usan el siguiente patrón:
- Contenedor padre con `position: relative`
- Imagen de fondo con `position: absolute` y `z-index: 1`
- Texto superpuesto con `position: relative` y `z-index: 2`
- Padding para centrar el texto sobre el contenedor visual

### 4. Eliminación de Elementos Obsoletos

Se eliminaron:
- Contenedores púrpuras antiguos
- Estilos de fondo sólido con `background-color`
- Bordes y sombras de caja CSS antiguos
- Borde amarillo del panel de apuestas (`border: 3px solid var(--gold-primary)`)
- Contenedores `pill-last-win` y `pill-total-win`
- Panel flotante `selected-animals-floating`
- CSS antiguo de `.animal-image-container` (borde dorado y fondo rojo generados por CSS - ahora comentado)
- CSS antiguo de `.animal-name-tag` (fondo rojo y borde blanco generados por CSS - ahora comentado)

## Beneficios de la Implementación

1. **Consistencia Visual**: Todos los displays usan el mismo estilo de contenedor
2. **Temática Mejorada**: Los assets PNG proporcionan una estética más cohesiva
3. **Flexibilidad**: Fácil cambiar el diseño modificando solo los archivos PNG
4. **Rendimiento**: Uso eficiente de assets reutilizables
5. **Mantenibilidad**: Código CSS más limpio y organizado
