# 3.1 Panel de Historial de Apuestas

**Fecha de implementación:** 5 de Diciembre de 2025
**Prioridad:** Media
**Estado:** ✅ Completado
**Categoría:** Feature / UI Enhancement

---

## Descripción

Sistema completo de historial de apuestas que registra todas las rondas jugadas, mostrando estadísticas detalladas y permitiendo al usuario revisar su desempeño histórico. El panel se presenta en un modal similar al tutorial y configuración, accesible mediante un botón en el header.

---

## Características Implementadas

### 1. **Interfaz de Datos (`bet-history.interface.ts`)**

```typescript
export interface BetEntry {
  animal: Animal;
  amount: number;
}

export interface BetHistory {
  id: number;                    // ID único del registro
  timestamp: Date;               // Fecha y hora de la apuesta
  bets: BetEntry[];             // Lista de apuestas realizadas
  totalBet: number;             // Total apostado en la ronda
  winningAnimal: Animal;        // Animal ganador
  winningMultiplier: number;    // Multiplicador del animal ganador
  winAmount: number;            // Cantidad ganada (0 si perdió)
  isWin: boolean;               // Si ganó o perdió
  balanceAfter: number;         // Balance después de la ronda
}
```

### 2. **Componente de Historial (`bet-history.component.ts`)**

**Propiedades:**
- `@Input() isOpen`: Controla la visibilidad del modal
- `@Input() historyData`: Array de registros históricos
- `@Output() closeModal`: Emite evento para cerrar el modal

**Getters Computados:**
- `sortedHistory`: Ordena el historial por fecha (más reciente primero)
- `statistics`: Calcula estadísticas agregadas:
  - Total de rondas jugadas
  - Total apostado
  - Total ganado
  - Porcentaje de victorias (win rate)
  - Ganancia/pérdida neta

**Métodos:**
- `formatDate(date: Date)`: Formatea fecha a formato local legible
- `onClose()`: Emite evento de cierre del modal

**Métodos implementados:**

#### `loadBetHistory(): void`
- Carga el historial desde localStorage al iniciar
- Convierte strings de fecha a objetos Date
- Actualiza el contador de IDs para evitar duplicados

#### `saveBetHistory(): void`
- Guarda el historial en localStorage
- Manejo de errores con try/catch

#### `addToHistory(winningAnimal, winningMultiplier, winAmount, isWin): void`
- Crea entrada de historial después de cada giro
- Filtra apuestas con amount > 0
- Incluye timestamp, balance final y toda la información de la ronda
- Agrega al inicio del array (más reciente primero)
- Guarda automáticamente en localStorage

#### `openBetHistory(): void`
- Muestra el modal de historial

#### `closeBetHistory(): void`
- Cierra el modal de historial

#### `clearBetHistory(): void`
- Borra todo el historial con confirmación del usuario
- Reinicia el contador de IDs
- Guarda cambios en localStorage

**Integración en `spinWheels()`:**
```typescript
// Agregar resultado al historial
this.addToHistory(
  result.animal,
  result.number,
  this.lastWin,
  this.lastWin > 0
);
```

## Persistencia de Datos

### LocalStorage
- **Key**: `'betHistory'`
- **Formato**: JSON string del array de BetHistory
- **Carga**: Al iniciar la aplicación (constructor)
- **Guardado**: Automático después de cada ronda
- **Conversión**: Strings de fecha se convierten a Date objects al cargar

### Estructura de Datos Guardada:
```json
[
  {
    "id": 0,
    "timestamp": "2025-12-05T10:30:00.000Z",
```

## Flujo de Uso

### Usuario Normal:
1. Usuario realiza apuestas y gira la ruleta
2. Sistema registra automáticamente el resultado en el historial
3. Usuario hace clic en el botón 📊 en el header
4. Se abre el modal con el historial completo
5. Usuario puede revisar:
   - Listado detallado de todas las rondas
   - Fecha, apuestas y resultados de cada ronda
6. Usuario cierra el modal haciendo clic en X o fuera del modal

### Datos Mostrados por Entrada:
- **Header de entrada**: Fecha/hora + Badge (Victoria/Derrota)
- **Sección de apuestas**: Lista de animales apostados con montos
- **Resultado**: Animal ganador + multiplicador
- **Footer**: Total apostado, ganancia, balance final

## Estadísticas Calculadas

### Total de Rondas
```typescript
totalRounds = this.historyData.length
```

### Victorias
```typescript
wins = this.historyData.filter(h => h.isWin).length
```

### Total Apostado
```typescript
totalBet = this.historyData.reduce((sum, h) => sum + h.totalBet, 0)
```

### Total Ganado
```typescript
totalWon = this.historyData.reduce((sum, h) => sum + h.winAmount, 0)
```

### Tasa de Victoria
```typescript
winRate = (wins / totalRounds) * 100  // Redondeado a entero
```

### Ganancia Neta
```typescript
netProfit = totalWon - totalBet
```

## Validaciones

✅ **Solo apuestas válidas**: Solo se guardan apuestas con amount > 0
✅ **IDs únicos**: Sistema de contador incremental previene duplicados
✅ **Formato de fecha**: Conversión correcta entre Date y string JSON
✅ **Ordenamiento**: Historial siempre ordenado por fecha descendente
✅ **Manejo de errores**: Try/catch en operaciones de localStorage
✅ **Estado vacío**: Mensaje amigable cuando no hay historial
