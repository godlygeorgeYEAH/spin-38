# Documentación 2.2 - Nombres de Animales en Rueda

## Fecha: 2025-12-06

## Descripción
Implementación de texto curvado para mostrar los nombres de los animales en la ruleta, siguiendo la curvatura de los segmentos con estilo blanco y contorno negro.

## Cambios Implementados

### 1. HTML - Texto Curvado en SVG

**Archivo**: [wheel-container.component.html](src/app/components/wheel-container/wheel-container.component.html) (líneas 66-79)

```html
<g *ngFor="let item of displayItems; let i = index">
  <defs>
    <path [attr.id]="'textPath-' + i + '-' + componentId"
          [attr.d]="getTextPathArc(i)"
          fill="none" />
  </defs>
  <text class="animal-name-text">
    <textPath [attr.href]="'#textPath-' + i + '-' + componentId"
              startOffset="50%"
              text-anchor="middle">
      {{ item.name }}
    </textPath>
  </text>
</g>
```

**Características:**
- Usa `<textPath>` de SVG para curvar el texto
- Un path único por cada segmento (`textPath-{i}-{componentId}`)
- Texto centrado con `startOffset="50%"` y `text-anchor="middle"`
- Se renderiza para cada animal en `displayItems`

### 2. TypeScript - Constante de Posicionamiento

**Archivo**: [wheel-container.component.ts](src/app/components/wheel-container/wheel-container.component.ts) (líneas 119-127)

```typescript
/**
 * Ratio para posicionar el texto curvado de nombres de animales.
 * El texto se posiciona al 88% del radio del anillo exterior.
 */
private readonly ANIMAL_TEXT_POSITION_RATIO = 0.88;
```

**Sistema de Posicionamiento:**
- Ratio del radio del anillo exterior: `0.88` (88%)
- Cerca del borde externo de los segmentos rojos
- Ajustable similar a `ANIMAL_POSITION_RATIO` (0.700) y `NUMBER_POSITION_RATIO` (0.450)

### 3. TypeScript - Generación del Arco

**Archivo**: [wheel-container.component.ts](src/app/components/wheel-container/wheel-container.component.ts) (líneas 602-621)

```typescript
public getTextPathArc(index: number): string {
  const textRadius = this.outerRingRadius * this.ANIMAL_TEXT_POSITION_RATIO;
  const angleStep = (2 * Math.PI) / this.segmentsCount;
  const startAngle = index * angleStep - Math.PI / 2;
  const endAngle = startAngle + angleStep;

  const x1 = textRadius * Math.cos(startAngle);
  const y1 = textRadius * Math.sin(startAngle);
  const x2 = textRadius * Math.cos(endAngle);
  const y2 = textRadius * Math.sin(endAngle);

  return `M ${x1},${y1} A ${textRadius},${textRadius} 0 0,1 ${x2},${y2}`;
}
```

**Funcionamiento:**
- Calcula arco SVG que sigue la curvatura del segmento
- Usa misma lógica angular que `getSegmentPath()`
- Radio dinámico basado en `ANIMAL_TEXT_POSITION_RATIO`
- Retorna path SVG en formato `M x1,y1 A radius,radius 0 0,1 x2,y2`

## Configuración de Posición

Para ajustar la posición del texto, modificar `ANIMAL_TEXT_POSITION_RATIO`:

```typescript
// Más cerca del borde exterior
private readonly ANIMAL_TEXT_POSITION_RATIO = 0.92;

// Más hacia el centro
private readonly ANIMAL_TEXT_POSITION_RATIO = 0.80;

// Sobre las imágenes de animales
private readonly ANIMAL_TEXT_POSITION_RATIO = 0.70;
```

**Valores recomendados:**
- `0.85` - `0.90`: Cerca del borde (legible)
- `0.75` - `0.80`: Entre imágenes y borde
- `0.65` - `0.70`: Sobre las imágenes

## Comportamiento

- **Visibilidad**: Siempre visible en la ruleta
- **Rotación**: El texto rota junto con la rueda
- **Curvatura**: Sigue la forma circular del segmento
- **Centrado**: Automáticamente centrado en cada segmento
