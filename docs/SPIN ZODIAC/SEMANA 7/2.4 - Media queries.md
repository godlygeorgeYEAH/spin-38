## Estructura de Media Queries

### Sistema de Variables CSS
Todas las resoluciones usan la variable `--wheel-diameter` para controlar el tamaño de la rueda de forma centralizada:

```css
:root {
  --wheel-diameter: 350px;  /* Cambiar solo este valor */
}

.wheel-container-wrapper,
::ng-deep .wheel-wrapper {
  width: var(--wheel-diameter) !important;
  height: var(--wheel-diameter) !important;
}
```

**Nota**: Se usa `::ng-deep` para penetrar el ViewEncapsulation de Angular y afectar los estilos del componente `app-wheel-container`.

## Correcciones Importantes

### Fix: Top Marker Rotation en Móviles
**Problema**: El `.top-marker` rotaba incorrectamente en resoluciones móviles porque el `transform: scale(0.9)` sobrescribía las transformaciones originales.

**Archivo**: [wheel-container.component.css:432-438](src/app/components/wheel-container/wheel-container.component.css#L432-L438)

**Antes**:
```css
.top-marker,
.center-needle {
  transform: scale(0.9);  /* ❌ Eliminaba translate y rotate */
}
```

**Después**:
```css
.top-marker {
  transform: translate(-50%, -50%) rotate(45deg) scale(0.9);  /* ✅ */
}

.center-needle {
  transform: scale(0.9);
}
```
