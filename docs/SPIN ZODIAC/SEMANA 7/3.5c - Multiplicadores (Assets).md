# Documentación 3.5c - Multiplicadores (Assets)

## Fecha: 2025-12-06

## Descripción
Implementación de badges visuales de multiplicadores en el overlay de victoria usando assets PNG.

## Assets Utilizados

**Ubicación**: `assets/images/multiplicadores/`
- `X1.png` - Multiplicador x1
- `X1.5.png` - Multiplicador x1.5
- `X2.png` - Multiplicador x2
- `X3.png` - Multiplicador x3
- `X5.png` - Multiplicador x5
- `X10.png` - Multiplicador x10

## Implementación

### HTML - Badge en Overlay de Victoria

**Archivo**: [home.page.html](src/app/home/home.page.html) (líneas 99-101)

```html
<img [src]="'assets/images/multiplicadores/X' + gameResult.number + '.png'"
     [alt]="'X' + gameResult.number"
     class="result-multiplier-badge">
```

**Características:**
- Ruta dinámica construida con `gameResult.number`
- Se muestra dentro del `.result-animal-container`
- Solo visible en victorias (`lastWin > 0`)

### CSS - Posicionamiento del Badge

**Archivo**: [home.page.css](src/app/home/home.page.css) (líneas 579-589)

```css
.result-multiplier-badge {
  position: absolute;
  top: 5px;
  right: 5px;
  width: 70px;
  height: 70px;
  object-fit: contain;
  z-index: 3;
  filter: drop-shadow(0 4px 8px rgba(0, 0, 0, 0.4));
}
```

**Posicionamiento:**
- Esquina superior derecha del aro dorado
- `top: 5px` y `right: 5px` para atravesar ligeramente el borde
- `z-index: 3` para estar sobre el aro y animal
- Sombra para profundidad visual

## Comportamiento

- **Condición**: Solo se muestra cuando `lastWin > 0` (victoria)
- **Multiplicador mostrado**: Corresponde a `gameResult.number`
- **Ubicación**: Overlay de resultados, sobre el aro del animal ganador
