# Documentación 3.5c - Botones (Assets)

## Fecha: 2025-12-05

## Descripción General
Implementación del sistema de botones usando assets PNG para reemplazar botones generados por CSS.

## Cambios Implementados

### 1. Sistema de Botones con Assets

#### Archivos de Assets Utilizados
- `assets/images/botones/limpiar.png` - Botón para limpiar todas las apuestas

#### Botones Implementados

**Archivo**: [home.page.html](src/app/home/home.page.html)

1. **Botón Limpiar** (línea 57)
   - Ubicación: Panel de acciones en betting-panel
   - Asset: `limpiar.png`
   - Función: Elimina todas las apuestas activas
   ```html
   <button (click)="clearAllBets()" class="clear-button"
     [disabled]="selectedAnimals.length === 0 || gameState === 'playing'">
     <img src="assets/images/botones/limpiar.png" alt="Limpiar" class="clear-button-image">
   </button>
   ```

### 2. Estructura CSS

**Archivo**: [home.page.css](src/app/home/home.page.css)

#### Estilos del Botón Limpiar (líneas 646-685)
```css
.clear-button {
  position: relative;
  border: none;
  background: transparent;
  cursor: pointer;
  padding: 0;
  min-width: 100px;
  height: 45px;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: opacity 0.2s, transform 0.2s;
}

.clear-button-image {
  position: absolute;
  width: 100%;
  height: 100%;
  object-fit: fill;
  z-index: 1;
}

.clear-button:hover:not(:disabled) {
  transform: scale(1.05);
}

.clear-button:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}
```

### 3. Sistema de Posicionamiento

- Botón con `position: relative`
- Imagen PNG con `position: absolute` y `object-fit: fill`
- Sin texto superpuesto - el asset incluye el texto
- Estados hover y disabled preservados

### 4. Eliminación de Elementos Obsoletos

Se eliminó (ahora comentado):
- CSS antiguo de `.clear-button` con fondo gris generado por CSS
- `background-color: #e0e0e0`
- `border-radius: 20px`
- `box-shadow: 0 2px 5px rgba(0, 0, 0, 0.3)`
- Hover con cambio de color de fondo

## Beneficios de la Implementación

1. **Consistencia Visual**: Botón usa el mismo patrón de assets PNG
2. **Diseño Personalizado**: Asset permite diseño completamente personalizado
3. **Flexibilidad**: Fácil cambiar diseño modificando solo el archivo PNG
4. **Mejor UX**: Efecto scale en hover más moderno que cambio de color
