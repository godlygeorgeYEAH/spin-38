# 3.3 Sistema de Creación de Monedas (Cambiar el Valor de las Fichas)

**Prioridad:** Media
**Estado:** ✅ Completado
**Categoría:** Nueva Funcionalidad / Configuración
**Fecha de Implementación:** 5 de Diciembre de 2025

---

## Descripción

Sistema simple de gestión de valores de fichas que permite a administradores autenticados modificar los valores de apuesta desde la consola del navegador. Los cambios se aplican inmediatamente en el juego sin necesidad de recargar la página.

---

## Características Implementadas

### 1. Gestión Dinámica de Fichas
- **Valores configurables:** Exactamente 6 fichas
- **Cambios en tiempo real:** Se aplican inmediatamente sin recargar
- **Validación completa:** Números positivos, sin duplicados, ordenamiento automático
- **Sin persistencia:** Al recargar página vuelven a valores por defecto

### 2. Comandos Disponibles

| Comando | Descripción | Requiere Auth |
|---------|-------------|---------------|
| `adminSetCoinValues([valores])` | Establecer 6 valores de fichas | ✅ Sí |
| `adminGetCoinValues()` | Ver valores actuales | ❌ No |
| `adminResetCoinValues()` | Resetear a valores por defecto | ✅ Sí |

### 3. Validaciones Implementadas

✅ **Cantidad:** Exactamente 6 fichas (no más, no menos)
✅ **Tipo:** Solo números enteros positivos
✅ **Duplicados:** No se permiten valores repetidos
✅ **Ordenamiento:** Se ordenan automáticamente de menor a mayor
✅ **Autenticación:** Cambios solo por admins autenticados

## Ejemplos de Validaciones

### ✅ Casos Válidos

```javascript
// 6 fichas - configuración estándar
adminSetCoinValues([1, 5, 10, 50, 100, 1000])
// ✅ Valores actualizados: [1, 5, 10, 50, 100, 1000]

// 6 fichas - valores personalizados
adminSetCoinValues([25, 50, 100, 250, 500, 2000])
// ✅ Valores actualizados: [25, 50, 100, 250, 500, 2000]

// Valores desordenados (se ordenan automáticamente)
adminSetCoinValues([1000, 10, 500, 100, 50, 5])
// ✅ Valores actualizados: [5, 10, 50, 100, 500, 1000]
```

### ❌ Casos Inválidos

```javascript
// Muy pocas fichas
adminSetCoinValues([10, 50, 100])
// ❌ Debes proporcionar exactamente 6 valores de fichas (proporcionaste 3)

// Demasiadas fichas
adminSetCoinValues([1, 5, 10, 20, 50, 100, 500])
// ❌ Debes proporcionar exactamente 6 valores de fichas (proporcionaste 7)

// Valores duplicados
adminSetCoinValues([10, 10, 50, 100])
// ❌ No se permiten valores duplicados

// Valores no numéricos
adminSetCoinValues([10, "50", 100])
// ❌ Todos los valores deben ser números enteros positivos

// Valores negativos
adminSetCoinValues([10, -50, 100])
// ❌ Todos los valores deben ser números enteros positivos

// Sin autenticación
adminSetCoinValues([10, 50, 100])
// ❌ Debes estar autenticado para cambiar valores de fichas
// 💡 Usa: adminLogin("admin", "contraseña")
```

### Archivos Modificados

**Comandos expuestos en `setupAdminCommands()`:**

```typescript
// Establecer valores
(window as any).adminSetCoinValues = (values: number[]) => {
  if (!this.adminAuth.isAuthenticated()) {
    console.log('❌ Debes estar autenticado...');
    return;
  }
  console.log(this.setCoinValues(values));
};

// Ver valores actuales
(window as any).adminGetCoinValues = () => {
  console.log(`💰 Valores actuales: [${this.coinValues.join(', ')}]`);
};

// Resetear valores
(window as any).adminResetCoinValues = () => {
  if (!this.adminAuth.isAuthenticated()) {
    console.log('❌ Debes estar autenticado...');
    return;
  }
  this.coinValues = [...this.DEFAULT_COIN_VALUES];
  this.cdr.detectChanges();
  console.log(`✅ Reseteado a: [${this.coinValues.join(', ')}]`);
};
```

**Método de validación `setCoinValues()`:**
```typescript
private setCoinValues(values: number[]): string {
  // Validación 1: Debe ser un array
  if (!Array.isArray(values)) {
    return '❌ Los valores deben ser un array...';
  }

  // Validación 2: Cantidad de fichas (3-6)
  if (values.length < 3 || values.length > 6) {
    return '❌ Debes proporcionar entre 3 y 6 valores de fichas';
  }

  // Validación 3: Solo números positivos
  if (!values.every(v => typeof v === 'number' && v > 0 && Number.isInteger(v))) {
    return '❌ Todos los valores deben ser números enteros positivos';
  }

  // Validación 4: Sin duplicados
  const uniqueValues = new Set(values);
  if (uniqueValues.size !== values.length) {
    return '❌ No se permiten valores duplicados';
  }

  // Ordenar y asignar
  this.coinValues = [...values].sort((a, b) => a - b);
  this.cdr.detectChanges();

  return `✅ Valores actualizados: [${this.coinValues.join(', ')}]`;
}
```

## Características del Sistema

### ✅ Ventajas

1. **Simple y Directo:**
   - No requiere servicios adicionales
   - Sin dependencias externas
   - Código mínimo y mantenible

2. **Cambios Inmediatos:**
   - Se aplican en tiempo real
   - No requiere recarga de página
   - Usa `ChangeDetectorRef` para actualizar UI

3. **Seguro:**
   - Requiere autenticación admin
   - Validación completa de entrada
   - Mensajes de error claros

4. **Flexible:**
   - Entre 3 y 6 fichas configurables
   - Cualquier valor entero positivo
   - Ordenamiento automático

### ⚠️ Limitaciones

1. **Sin Persistencia:**
   - Los cambios se pierden al recargar página
   - Vuelven a valores por defecto

2. **Local al Navegador:**
   - Cambios solo afectan la sesión actual
   - No se sincronizan entre dispositivos

**Nota:** Estas limitaciones son **intencionales** según los requisitos. El objetivo es facilitar que en el futuro un script inicialice los valores al cargar la app.
