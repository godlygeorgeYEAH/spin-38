**Ubicación**: [home.page.css:492-503](src/app/home/home.page.css#L492-L503)

### 2. Animal Image Container
**Clase**: `.animal-image-container`
**Posición**: Centro superior (sobresale del borde)

```css
.animal-image-container {
  width: 204px;
  height: 204px;
  position: absolute;
  top: 0;
  left: 50%;
  transform: translate(-50%, -50%);
  z-index: 10;

```

**Características**:
- Usa `aro-dorado.png` como marco
- Sobresale del borde superior (`top: 0` con `translate(-50%, -50%)`)
- Tamaño reducido 15% desde 240px original

**Ubicación**: [home.page.css:558-569](src/app/home/home.page.css#L558-L569)

### 3. Chips Section
**Clase**: `.chips-section`
**Posición**: Izquierda centro
**Grid**: 3 columnas × N filas

```css
.chips-section {
  --chips-scale: 1;
  position: absolute;
  left: 15px;
  top: 50%;
  transform: translateY(-50%) scale(var(--chips-scale));
  transform-origin: left center;
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 8px;
  width: 150px;
  z-index: 2;
}
```

**Parámetro de escala**: `--chips-scale`
- Valores sugeridos: `0.8` a `1.5`
- Mantiene siempre 3 columnas
- Escala desde el lado izquierdo y centro vertical

**Ubicación**: [home.page.css:466-478](src/app/home/home.page.css#L466-L478)

### 4. Player Balance Display

### Escala de Fichas
```css

  --chips-scale: 1;    /* Tamaño normal */
  --chips-scale: 1.2;  /* 20% más grande */
  --chips-scale: 0.8;  /* 20% más pequeño */
}
```

## Notas Técnicas

- **Position Absolute**: Todos los elementos usan `position: absolute` para control preciso
- **Transform Origin**: Las fichas escalan desde `left center` para mantener alineación
- **Pointer Events**: Total bet container usa `pointer-events: none` para no interferir con interacciones
- **CSS Variables**: `--chips-scale` permite ajuste dinámico del tamaño de fichas
