# 3.5b - Implementación de Fichas (Assets Visuales)

**Fecha de implementación:** 5 de Diciembre de 2025
**Prioridad:** Media
**Estado:** ✅ Completado
**Categoría:** Feature / Deprecación

---

## Descripción

Implementación del sistema de fichas visuales usando imágenes PNG de la carpeta `assets/images/fichas/`. Se deprecó el sistema anterior de fichas dibujadas con CSS y colores de fondo.

---

## Sistema de Numeración

Las fichas están numeradas del 1 al 6 según su orden en el array `coinValues`:

```
coin-bg-1.png → coinValues[0] → Menor denominación
coin-bg-2.png → coinValues[1]
coin-bg-3.png → coinValues[2]
coin-bg-4.png → coinValues[3]
coin-bg-5.png → coinValues[4]
coin-bg-6.png → coinValues[5] → Mayor denominación
```

**Regla importante:** La numeración siempre respeta el orden del array, independientemente de los valores de denominación.

## Archivos Modificados

### 1. **src/app/home/home.page.ts** (Líneas 454-463)
**Cambio:** Agregar método `getCoinImage()`

```typescript
/**
 * Obtiene la ruta de la imagen de la ficha basada en su valor
 * Las fichas están numeradas 1-6 según su orden en coinValues
 * coin-bg-1.png = coinValues[0], coin-bg-2.png = coinValues[1], etc.
 */
public getCoinImage(coinValue: number): string {
  const index = this.coinValues.indexOf(coinValue);
  if (index === -1) return '';
  return `assets/images/fichas/coin-bg-${index + 1}.png`;
}
```

## Sistema Deprecado

### Clases CSS Eliminadas:
- `.coin-chip` (contenedor con background CSS)
- `.chip-1`, `.chip-5`, `.chip-10`, `.chip-30`, `.chip-50`, `.chip-1000` (colores por denominación)
- Variables CSS: `--chip-color`

### Razón de Deprecación:
- Sistema anterior dibujaba fichas con colores CSS sólidos
- No utilizaba los assets visuales disponibles
- Menos flexible para personalización visual

---

## Ventajas del Nuevo Sistema

✅ **Assets visuales:** Usa imágenes PNG diseñadas específicamente
✅ **Flexibilidad:** Las imágenes pueden ser reemplazadas sin tocar código
✅ **Consistencia:** Numeración 1-6 siempre respeta el orden, sin importar denominaciones
✅ **Escalabilidad:** Cambiar valores de fichas no requiere modificar CSS
✅ **Profesional:** Apariencia visual más pulida

## Mapeo Actual (Ejemplo)

Con `coinValues = [1, 5, 10, 30, 50, 1000]`:

```
coin-bg-1.png → $1
coin-bg-2.png → $5
coin-bg-3.png → $10
coin-bg-4.png → $30
coin-bg-5.png → $50
coin-bg-6.png → $1000
```

Si se cambian los valores a `[10, 20, 50, 100, 500, 5000]`:

```
coin-bg-1.png → $10
coin-bg-2.png → $20
coin-bg-3.png → $50
coin-bg-4.png → $100
coin-bg-5.png → $500
coin-bg-6.png → $5000
```
